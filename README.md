# React Three Fiber Starter Project

## w/ the following packages:

- 🌐 @react-three/fiber
- 🌐 @react-three/drei
- 🌐 three
- 🪁 tailwindcss
- 💅 styled-components
- 🌋 leva

---

### Get Going

`git clone https://github.com/klingat/r3f-tailwind-starter.git your_folder_name`

`cd your_folder_name`

`yarn install`

`yarn run dev`
